# Keyword Output Template

Filename pattern:

`YYYYMMDD_HHMMSS_topic_kws.md`

Example filename:

`20260423_154500_vong-lap-python_kws.md`

File contents:

```md
| date | topic | main keyword | subkeyword |
|---|---|---|---|
| [YYYY-MM-DD HH:MM:SS] | [readable topic] | [MAIN_KEYWORD] | ["SUBKEYWORD_1", "SUBKEYWORD_2", "SUBKEYWORD_3", "SUBKEYWORD_4"] |
```
