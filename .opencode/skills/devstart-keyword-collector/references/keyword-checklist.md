# Keyword Checklist

## Context Check

- `.agents/product-marketing-context.md` was checked first if present
- Audience and terminology align with product context when available
- If no context exists, keyword choices stay neutral and educational

## Topic Check

- The topic is narrow enough for one page
- The topic matches this repo's educational content style
- The topic is suitable for `devstart-content-writing`

## Keyword Check

- Exactly one main keyword
- 5-12 subkeywords
- Subkeywords share the same search intent as the main keyword
- Off-topic keywords are excluded
- Main keyword can support title, slug, description, and one H2

## Output Check

- Output is markdown only
- Output uses a single table
- `subkeyword` column is in array format
- Filename follows `YYYYMMDD_HHMMSS_topic_kws.md`
- File is stored under `.agents/keyword-research/`
