# HTML CSS Course Roadmap

Muc tieu cua khoa hoc nay la dua nguoi moi tu viec hieu HTML co ban den muc tu tao duoc mot trang web don gian, co bo cuc ro rang va responsive tren man hinh nho.

## Nguyen tac xay dung

- Mot bai hoc = mot chu de ro rang, de dua vao `devstart-content-writing`
- Uu tien ket qua nhin thay duoc ngay tren trinh duyet
- Di tu cau truc HTML -> thanh phan thuong gap -> CSS nen tang -> bo cuc -> responsive
- Giu 2 bai HTML dang co va mo rong de khoa hoc thuc su bao trum ca HTML va CSS
