# Python Course Roadmap

Muc tieu cua khoa hoc nay la dua nguoi moi tu muc 0 den muc tu tin viet cac chuong trinh Python co ban va hoan thanh mot mini project thuc te.

## Nguyen tac xay dung

- Mot bai hoc = mot chu de ro rang, de dua vao `devstart-content-writing`
- Uu tien y dinh tim kiem cua nguoi moi, ngon ngu Viet, giai thich truc tiep
- Sap xep tu nen tang -> thao tac du lieu -> ky nang thuc chien -> du an
- Giu cac bai dang co, nhung trong roadmap tong the, bai vong lap nen dung sau bai dieu kien
