`FormApp` là service giúp Apps Script tạo và chỉnh sửa Google Forms bằng code. Bạn dùng nó khi cần tạo nhiều form giống cấu trúc, tạo form từ danh mục trong Sheets, chuẩn hóa câu hỏi đăng ký, hoặc tự động tạo khảo sát cho từng lớp, từng nhóm, từng sự kiện.

## FormApp trong Apps Script là gì?

Google Forms thường được tạo thủ công bằng giao diện. Cách đó ổn với một form nhỏ, nhưng bắt đầu bất tiện khi bạn cần tạo nhiều form có cấu trúc giống nhau hoặc danh sách lựa chọn thay đổi theo dữ liệu trong Sheets. `FormApp` cho phép script tạo form mới, thêm câu hỏi, đặt required, lấy edit URL, lấy published URL và cấu hình nơi lưu phản hồi.

Mô hình tư duy là: form là container, item là câu hỏi. Mỗi loại câu hỏi có method riêng như `addTextItem`, `addParagraphTextItem`, `addMultipleChoiceItem`, `addListItem`, `addCheckboxItem`. Sau khi tạo item, bạn chain method như `setTitle`, `setRequired`, `setHelpText`, `setChoiceValues`.

## Tạo form đăng ký đầu tiên

Ví dụ dưới đây tạo form đăng ký học với câu hỏi tên, email và khóa học. Code dùng tiêu đề ASCII để dễ copy, nhưng trong form thật bạn có thể dùng tiếng Việt.

```javascript
function createRegistrationForm() {
  const form = FormApp.create('Apps Script registration');

  form.setDescription('Registration form generated by Apps Script');

  form.addTextItem()
    .setTitle('Name')
    .setRequired(true);

  form.addTextItem()
    .setTitle('Email')
    .setRequired(true);

  form.addMultipleChoiceItem()
    .setTitle('Course')
    .setChoiceValues(['Apps Script basics', 'Apps Script automation'])
    .setRequired(true);

  Logger.log('Edit URL: ' + form.getEditUrl());
  Logger.log('Published URL: ' + form.getPublishedUrl());
}
```

`getEditUrl()` dành cho người chỉnh sửa form. `getPublishedUrl()` dành cho người trả lời. Nhầm hai URL này là lỗi phổ biến: gửi edit URL cho người ngoài có thể gây rủi ro quyền chỉnh sửa.

## Tạo câu hỏi từ danh sách trong Sheets

Use case mạnh hơn là lấy danh sách lựa chọn từ sheet. Ví dụ sheet `Courses` có danh sách khóa học ở cột A, bắt đầu từ dòng 2. Script đọc danh sách này và đưa vào dropdown.

```javascript
function createFormFromSheetOptions() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Courses');
  const options = sheet.getRange('A2:A').getValues()
    .flat()
    .filter(function(value) {
      return value;
    });

  if (options.length === 0) {
    throw new Error('No course options found');
  }

  const form = FormApp.create('Choose a course');

  form.addTextItem()
    .setTitle('Name')
    .setRequired(true);

  form.addTextItem()
    .setTitle('Email')
    .setRequired(true);

  form.addListItem()
    .setTitle('Course')
    .setChoiceValues(options)
    .setRequired(true);

  Logger.log(form.getPublishedUrl());
}
```

Đây là pattern tốt khi danh mục thay đổi thường xuyên. Team chỉ cần cập nhật sheet `Courses`, script tạo form mới theo dữ liệu hiện tại.

## Lưu form ID để cập nhật về sau

Nếu không lưu form ID, bạn chỉ có URL trong log và rất khó cập nhật form sau này. Hãy ghi ID vào PropertiesService hoặc sheet cấu hình.

```javascript
function createAndSaveFormId() {
  const form = FormApp.create('Saved form example');

  form.addTextItem()
    .setTitle('Email')
    .setRequired(true);

  PropertiesService.getScriptProperties()
    .setProperty('REGISTRATION_FORM_ID', form.getId());
}

function addQuestionToSavedForm() {
  const formId = PropertiesService.getScriptProperties().getProperty('REGISTRATION_FORM_ID');

  if (!formId) {
    throw new Error('Missing form ID');
  }

  const form = FormApp.openById(formId);
  form.addParagraphTextItem().setTitle('Notes');
}
```

Pattern này giúp bạn không tạo form mới liên tục khi thật ra chỉ muốn cập nhật form hiện có.

## Gắn form với Google Sheets để lưu phản hồi

Form có thể gửi phản hồi vào spreadsheet. Bạn có thể dùng `setDestination` với ID spreadsheet.

```javascript
function createFormWithResponseSheet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const form = FormApp.create('Lead capture form');

  form.addTextItem().setTitle('Name').setRequired(true);
  form.addTextItem().setTitle('Email').setRequired(true);

  form.setDestination(FormApp.DestinationType.SPREADSHEET, ss.getId());

  Logger.log('Form URL: ' + form.getPublishedUrl());
}
```

Sau khi có response sheet, bạn có thể tạo trigger `onFormSubmit` để xử lý dữ liệu ở bài tiếp theo.

## Khi nào dùng FormApp?

- Tạo nhiều form tương tự nhau cho nhiều lớp hoặc sự kiện.
- Tạo form từ danh mục trong Sheets.
- Chuẩn hóa câu hỏi bắt buộc thay vì tạo thủ công dễ thiếu.
- Tự động tạo khảo sát cho từng nhóm khách hàng.
- Tạo form đăng ký và lưu ID để quản lý bằng script.

Nếu chỉ tạo một form đơn giản và ít thay đổi, giao diện Google Forms có thể nhanh hơn. `FormApp` đáng dùng khi bạn cần lặp lại, chuẩn hóa hoặc tích hợp với dữ liệu.

## Lỗi thường gặp với FormApp

- Nhầm edit URL và published URL.
- Không đặt câu hỏi required nên dữ liệu phản hồi thiếu.
- Dùng tên câu hỏi không ổn định, gây lỗi khi đọc `namedValues`.
- Tạo form mới liên tục khi chỉ muốn cập nhật form cũ.
- Không lưu form ID sau khi tạo.
- Không kiểm tra danh sách lựa chọn rỗng trước khi gọi `setChoiceValues`.
- Quên gắn form với spreadsheet phản hồi nếu muốn xử lý bằng Sheets.

## Câu hỏi thường gặp về FormApp

### Có thể tạo quiz bằng Apps Script không?

Có, `FormApp` hỗ trợ một số thiết lập quiz và feedback. Tuy nhiên người mới nên thành thạo form thường, item, choices và response destination trước.

### Có thể gắn form với Sheets không?

Có. Dùng `setDestination(FormApp.DestinationType.SPREADSHEET, spreadsheetId)` để lưu phản hồi vào spreadsheet.

### Có thể sửa form đã có không?

Có. Dùng `FormApp.openById(formId)` nếu bạn có ID và quyền chỉnh sửa form.

### Có nên tạo form mới mỗi lần chạy không?

Không nên nếu mục tiêu là cập nhật form cũ. Hãy lưu form ID và mở lại bằng `openById`.

## Tóm tắt và chuyển tiếp

`FormApp` giúp tạo Google Forms tự động, đặc biệt hữu ích khi form cần lấy lựa chọn từ Sheets hoặc cần tạo nhiều lần theo cùng cấu trúc. Hãy lưu form ID, phân biệt edit URL và published URL, và gắn form với Sheets nếu muốn xử lý phản hồi. Bài tiếp theo sẽ học cách kết nối Forms với Sheets và xử lý submit bằng trigger.
