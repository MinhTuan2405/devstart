## Python là gì?

Python là ngôn ngữ lập trình bậc cao, được tạo ra bởi Guido van Rossum và ra mắt lần đầu vào năm 1991. Python nổi tiếng vì cú pháp đơn giản, dễ đọc — gần giống tiếng Anh tự nhiên.

Python được sử dụng rộng rãi trong nhiều lĩnh vực:

- **Phát triển web** (Django, Flask)
- **Khoa học dữ liệu** (Pandas, NumPy)
- **Trí tuệ nhân tạo** (TensorFlow, PyTorch)
- **Tự động hóa** (scripting, bot)
- **Phát triển game** (Pygame)

## Tại sao nên học Python?

1. **Dễ học**: Cú pháp đơn giản, ít ký tự đặc biệt
2. **Cộng đồng lớn**: Hàng triệu lập trình viên, rất dễ tìm tài liệu
3. **Nhiều việc làm**: Python luôn nằm trong top ngôn ngữ được tuyển dụng nhiều nhất
4. **Đa năng**: Dùng được cho web, data, AI, automation...

## Cài đặt Python

Truy cập [python.org](https://python.org) và tải bản Python mới nhất.

**Trên Windows:**
1. Tải file `.exe` từ python.org
2. Chạy installer, **nhớ tick "Add Python to PATH"**
3. Mở Command Prompt, gõ `python --version` để kiểm tra

**Trên macOS:**
1. Mở Terminal
2. Cài qua Homebrew: `brew install python`

## Chương trình đầu tiên

Mở terminal hoặc IDLE, gõ:

```python
print("Xin chào, mình là Python!")
```

Kết quả:

```
Xin chào, mình là Python!
```

Chúc mừng! Bạn vừa viết dòng code đầu tiên bằng Python. Ở bài tiếp theo, chúng ta sẽ tìm hiểu về **biến và kiểu dữ liệu** trong Python.
