## Tuple trong Python là gì?

Tuple trong Python là kiểu dữ liệu dùng để lưu nhiều giá trị giống như list, nhưng điểm khác biệt quan trọng là tuple không thể thay đổi sau khi tạo. Vì vậy, tuple rất hợp khi bạn muốn giữ dữ liệu ổn định như tọa độ, ngày tháng, hoặc cấu hình cố định.

Nếu list phù hợp cho dữ liệu hay cập nhật, thì tuple phù hợp cho dữ liệu cần “khóa lại”.

## Cú pháp cơ bản của tuple trong Python

Tuple thường được viết trong dấu `()`.

```python
toa_do = (10, 20)
mon_hoc = ("Python", "HTML", "C++")

print(toa_do)
print(mon_hoc[0])
print(len(mon_hoc))
```

Bạn vẫn có thể truy cập phần tử bằng chỉ số giống như list.

```python
thong_tin = ("Lan", 20, "Ha Noi")

print(thong_tin[1])
print(thong_tin[-1])
```

## Cách dùng tuple trong Python cho dữ liệu cố định

Tuple rất hay được dùng khi dữ liệu không nên bị sửa trong lúc chạy chương trình.

```python
mau_rgb = (255, 0, 0)

print(f"Mau do co gia tri: {mau_rgb}")
```

Ngoài ra, tuple cũng thường được dùng khi một hàm trả về nhiều giá trị cùng lúc.

```python
def tinh_hieu_va_tong(a, b):
    return (a - b, a + b)

ket_qua = tinh_hieu_va_tong(10, 3)
print(ket_qua)
```

## Tuple và list trong Python khác nhau thế nào?

- `list` dùng dấu `[]` và có thể thay đổi.
- `tuple` dùng dấu `()` và không thể thay đổi.
- `list` phù hợp khi bạn cần thêm, sửa, xóa dữ liệu.
- `tuple` phù hợp khi dữ liệu cần ổn định.

Ví dụ lỗi thường gặp:

```python
so = (1, 2, 3)

# Sai vì tuple khong cho phep thay doi phan tu
# so[0] = 10

print(so)
```

## Những lỗi thường gặp với tuple trong Python

- Nghĩ rằng tuple có thể sửa trực tiếp như list.
- Quên dấu phẩy khi tạo tuple chỉ có một phần tử.
- Nhầm tuple với list vì cả hai đều lưu được nhiều giá trị.

Ví dụ tuple một phần tử:

```python
a = (5,)
print(type(a))
```

Nếu viết `(5)` thì đó chỉ là số `5` trong ngoặc, chưa phải tuple.

## Câu hỏi thường gặp về tuple trong Python

### Vì sao tuple trong Python không sửa được?

Tuple được thiết kế cho dữ liệu cố định để tránh bị thay đổi ngoài ý muốn. Điều này giúp code an toàn hơn trong những trường hợp bạn muốn giữ nguyên giá trị ban đầu.

### Khi nào nên dùng tuple thay vì list?

Hãy dùng tuple khi dữ liệu không cần thêm, sửa, xóa sau khi tạo. Ví dụ phổ biến là tọa độ, thông số cấu hình, hoặc bộ dữ liệu trả về từ hàm.
