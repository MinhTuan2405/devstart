## Thẻ tiêu đề (Heading)

HTML có 6 cấp tiêu đề, từ `h1` (lớn nhất) đến `h6` (nhỏ nhất):

```html
<h1>Tiêu đề cấp 1</h1>
<h2>Tiêu đề cấp 2</h2>
<h3>Tiêu đề cấp 3</h3>
```

Mỗi trang chỉ nên có **một thẻ h1** duy nhất (quan trọng cho SEO).

## Đoạn văn và định dạng

```html
<p>Đây là một đoạn văn.</p>
<p>Chữ <strong>in đậm</strong> và chữ <em>in nghiêng</em>.</p>
<p>Xuống dòng bằng thẻ <br> tự đóng.</p>
```

## Liên kết (Link)

```html
<a href="https://devstart.vn">Truy cập DevStart</a>
<a href="/khoa-hoc">Xem khóa học</a>
<a href="https://google.com" target="_blank">Mở tab mới</a>
```

## Hình ảnh

```html
<img src="hinh-anh.jpg" alt="Mô tả hình ảnh" width="300">
```

Thuộc tính `alt` rất quan trọng cho SEO và accessibility.

## Danh sách

Danh sách không thứ tự:

```html
<ul>
    <li>Python</li>
    <li>C++</li>
    <li>HTML/CSS</li>
</ul>
```

Danh sách có thứ tự:

```html
<ol>
    <li>Bước 1: Cài đặt</li>
    <li>Bước 2: Viết code</li>
    <li>Bước 3: Chạy thử</li>
</ol>
```

Bài tiếp theo sẽ giới thiệu về **CSS** — cách để trang web đẹp hơn!
