## HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để tạo cấu trúc cho trang web. Mọi trang web bạn thấy trên internet đều được xây dựng bằng HTML.

HTML không phải ngôn ngữ lập trình — nó là ngôn ngữ **đánh dấu** (markup), dùng các **thẻ (tags)** để mô tả nội dung.

## Cấu trúc cơ bản

```html
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang web đầu tiên</title>
</head>
<body>
    <h1>Xin chào!</h1>
    <p>Đây là trang web đầu tiên của tôi.</p>
</body>
</html>
```

## Giải thích từng phần

- `<!DOCTYPE html>`: Khai báo đây là tài liệu HTML5
- `<html>`: Thẻ gốc chứa toàn bộ nội dung
- `<head>`: Chứa thông tin meta (tiêu đề, charset...)
- `<body>`: Chứa nội dung hiển thị trên trang
- `<h1>`: Tiêu đề cấp 1 (lớn nhất)
- `<p>`: Đoạn văn bản

## Thực hành

1. Tạo file `index.html`
2. Copy đoạn code phía trên
3. Mở file bằng trình duyệt (Chrome, Firefox)

Chúc mừng! Bạn vừa tạo trang web đầu tiên. Bài tiếp theo sẽ giới thiệu **các thẻ HTML cơ bản** mà bạn cần biết.
