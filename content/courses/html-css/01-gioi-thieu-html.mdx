## HTML là gì?

HTML là ngôn ngữ đánh dấu dùng để tạo cấu trúc cho trang web. Nếu bạn đang tìm câu trả lời ngắn gọn cho "HTML là gì", thì HTML là lớp nền giúp trình duyệt hiểu đâu là tiêu đề, đoạn văn, hình ảnh, liên kết, danh sách và các phần nội dung khác trên một trang.

HTML không phải ngôn ngữ lập trình. Nó không có biến, vòng lặp hay hàm như JavaScript, Python hoặc C++. Thay vào đó, HTML dùng các **thẻ** để đánh dấu ý nghĩa của nội dung. Nói dễ hiểu, nếu website là một ngôi nhà thì HTML là bộ khung và cách chia phòng.

Khi bắt đầu học HTML cho người mới, bạn chưa cần nhớ hàng chục thẻ ngay. Điều quan trọng nhất là hiểu HTML dùng để làm gì: tạo cấu trúc, giúp trình duyệt hiển thị nội dung đúng cách và làm nền để CSS tạo giao diện đẹp hơn.

## Cấu trúc cơ bản

```html
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang web đầu tiên</title>
</head>
<body>
    <h1>Xin chào!</h1>
    <p>Đây là trang web đầu tiên của tôi.</p>
</body>
</html>
```

Đây là ví dụ HTML cơ bản gần như ai học web cũng gặp đầu tiên. Dù trang rất ngắn, nó đã có đủ phần đầu và phần nội dung hiển thị.

## Giải thích từng phần

- `<!DOCTYPE html>`: Khai báo đây là tài liệu HTML5
- `<html>`: Thẻ gốc chứa toàn bộ nội dung
- `<head>`: Chứa thông tin meta (tiêu đề, charset...)
- `<body>`: Chứa nội dung hiển thị trên trang
- `<h1>`: Tiêu đề cấp 1 (lớn nhất)
- `<p>`: Đoạn văn bản

### Vì sao phải có `head` và `body`?

Người mới thường chỉ chú ý phần chữ hiển thị trên màn hình, nhưng trình duyệt cần nhiều thông tin hơn thế.

- `head` chứa dữ liệu hỗ trợ như tiêu đề tab, bảng mã tiếng Việt, mô tả trang, file CSS.
- `body` mới là phần người dùng thực sự nhìn thấy.

Nếu thiếu `meta charset="UTF-8"`, chữ tiếng Việt có thể bị lỗi font hoặc hiển thị sai dấu. Đây là lỗi rất phổ biến khi tạo trang web đầu tiên bằng HTML.

### Thẻ mở và thẻ đóng hoạt động ra sao?

Phần lớn thẻ HTML có dạng mở và đóng:

```html
<p>Xin chào các bạn</p>
```

Trong đó:

- `<p>` là thẻ mở
- `</p>` là thẻ đóng
- `Xin chào các bạn` là nội dung nằm bên trong

Một số thẻ đặc biệt như `meta` hoặc `img` không cần thẻ đóng theo cách thông thường, nhưng lúc mới học bạn chỉ cần nắm chắc các thẻ có cặp mở đóng trước.

## HTML dùng để làm gì trong thực tế?

Rất nhiều bạn hỏi HTML có phải ngôn ngữ lập trình không và học HTML để làm gì. Câu trả lời thực tế là:

- Tạo trang giới thiệu cá nhân
- Tạo bài viết blog đơn giản
- Tạo trang sản phẩm hoặc landing page
- Tạo form đăng ký, liên hệ, đặt lịch
- Làm nền tảng để học CSS và JavaScript

Ngay cả khi sau này bạn dùng React, Next.js hay một framework hiện đại, phần giao diện cuối cùng vẫn được trình duyệt hiểu dưới dạng HTML.

## Ví dụ HTML cơ bản dễ hiểu hơn

Ví dụ dưới đây cho thấy một trang có tiêu đề, mô tả và liên kết:

```html
<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <title>Gioi thieu ban than</title>
</head>
<body>
  <h1>Xin chao, minh la An</h1>
  <p>Minh dang hoc HTML de tu tao trang web dau tien.</p>
  <a href="https://example.com">Xem them</a>
</body>
</html>
```

Khi mở file này trên trình duyệt, bạn sẽ thấy rõ HTML không "làm đẹp" nhiều, nhưng nó tổ chức nội dung rất rõ ràng. Việc làm đẹp sẽ đến ở bài CSS.

## Cách tạo trang web đầu tiên bằng HTML

Nếu bạn chưa từng viết code, hãy làm thật chậm theo đúng thứ tự:

- Tạo một thư mục mới, ví dụ `trang-web-dau-tien`
- Tạo file `index.html`
- Dán mẫu HTML cơ bản vào file
- Lưu lại
- Nhấp đúp để mở bằng trình duyệt

Khi sửa chữ trong `h1` hoặc `p`, chỉ cần lưu file rồi tải lại trình duyệt là bạn sẽ thấy kết quả. Cách học này rất hiệu quả vì bạn nhìn được thay đổi ngay lập tức.

## Những lỗi thường gặp khi bắt đầu học HTML

- Quên thẻ đóng như quên `</p>` hoặc `</body>`
- Viết sai tên thẻ, ví dụ `htlm` thay vì `html`
- Quên `<!DOCTYPE html>` nên trình duyệt có thể hiển thị khác chuẩn
- Quên `meta charset="UTF-8"` làm lỗi tiếng Việt
- Lưu sai đuôi file, ví dụ `.txt` thay vì `.html`

Ví dụ sai thường gặp:

```html
<html>
<head>
  <title>Trang dau tien</title>
</head>
<body>
  <h1>Xin chao
  <p>Toi dang hoc HTML.
</body>
</html>
```

Ở ví dụ trên, `h1` và `p` đều chưa được đóng. Nhiều trình duyệt vẫn cố hiển thị, nhưng code như vậy khó đọc và dễ gây lỗi khi trang lớn hơn.

## Bài tập thực hành

Hãy tự tạo một file `index.html` với các yêu cầu sau:

- Có tiêu đề tab là tên của bạn
- Có một `h1` giới thiệu bản thân
- Có hai đoạn văn mô tả bạn đang học gì
- Có một liên kết tới trang web bạn thích

Mục tiêu của bài tập không phải làm trang đẹp, mà là giúp bạn quen tay với cấu trúc trang HTML cơ bản và thói quen đóng thẻ cẩn thận.

## Câu hỏi thường gặp

### HTML có phải ngôn ngữ lập trình không?

Không. HTML là ngôn ngữ đánh dấu. Nó giúp mô tả cấu trúc nội dung trên trang, chứ không xử lý logic như ngôn ngữ lập trình.

### Chỉ học HTML có làm được website không?

Có thể làm được trang rất cơ bản. Nhưng để giao diện đẹp và có tương tác, bạn sẽ cần thêm CSS và sau đó có thể là JavaScript.

### Vì sao file đầu tiên thường tên là `index.html`?

Đó là tên mặc định rất phổ biến cho trang chính. Nhiều máy chủ web và framework sẽ tự nhận `index.html` là trang đầu tiên.

Bạn đã biết HTML là gì, HTML dùng để làm gì, cấu trúc trang HTML cơ bản gồm những phần nào, và cách tạo trang web đầu tiên bằng HTML. Nền tảng quan trọng nhất ở giai đoạn này là hiểu vai trò của từng thẻ và viết code gọn, đúng cấu trúc.

Ở bài tiếp theo, chúng ta sẽ đi vào **các thẻ HTML cơ bản** như tiêu đề, đoạn văn, liên kết, hình ảnh và danh sách để bạn viết nội dung phong phú hơn.
