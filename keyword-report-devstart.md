# DevStart

## Giới thiệu website

Các nhóm nội dung chính của website gồm:

- Khóa học Python
- Khóa học C++
- Khóa học HTML/CSS
- Khóa học SQL
- Khóa học Machine Learning
- Khóa học Claude Code
- Blog hướng dẫn lập trình và công nghệ
- Bộ câu hỏi phỏng vấn lập trình theo chủ đề
